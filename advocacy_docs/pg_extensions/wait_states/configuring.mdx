## Parameters

The following parameters control the EDB Wait States extension behavior.  If you modify these parameters, reload Postgres to apply the changes.

| Parameter                    | Description | Default |                                                                                                                                                                                                                                                  |
|---------------------------------|-------------|---------|
| `edb_wait_states.sampling_interval`| The interval between two EDB Wait States sampling cycles.| 1 second |
| `edb_wait_states.retention_period`| Deletes EDB Wait States log files after a defined retention period. | 604800 seconds (7 days)|                                                                                                                                                                               |
| `edb_wait_states.enable_collection`| Enable or disable EDB Wait States data collection. | True |                                                                                                                                                                                                                                         |
| `edb_wait_states.directory` | Stores the EDB Wait States logs in this directory. The path must be a full, absolute path.  It can't be a relative path. | $PGDATA/edb_wait_states |
