Transparent data encryption (TDE) is an optional feature supported by EDB Postgres Advanced Server and EDB Postgres Extended Server from version 15. 

It encrypts any user data stored in the database system. Review these topics to learn more:

- [Why should you use TDE?](why)
- [What's encrypted with TDE?](what)
- [How does TDE encrypt data?](how)
