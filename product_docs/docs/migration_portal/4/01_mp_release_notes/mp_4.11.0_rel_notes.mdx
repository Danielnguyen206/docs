Released: 20 Aug 2024

New features, enhancements, bug fixes, and other changes in Migration Portal 4.11.0 include the following:

| Type              | Description                                                                                                      |
|-------------------|------------------------------------------------------------------------------------------------------------------|
| Enhancement       | Updated the styling for logos, fonts and layouts for an improved look and feel.                                  |
| Security&nbsp;fix | Updated software libraries and implemented other security improvements.                                          |
| Bug&nbsp;fix      | Fixed the 'relation does not exist' error for public synonyms.                                                   |
| Bug&nbsp;fix      | Fixed an issue where errors were being reported for compatible `MERGE` and `UNISTR` statements.                  |
| Bug&nbsp;fix      | Fixed an issue where the report generation took a long time when a project contained a large number of objects.  |
| Bug&nbsp;fix      | Fixed an auto-scrolling issue observed on AI Copilot when changing tabs.                                         |
| Bug&nbsp;fix      | Fixed an issue that prevented the usage of the "@" sign in database link names.                                  |
| Bug&nbsp;fix      | Fixed an issue where database link names containing multiple dots were quoted incorrectly.                       |
| Bug&nbsp;fix      | Fixed an issue that caused the Migration Portal to take a long time loading a project after a schema assessment. |
| Bug&nbsp;fix      | Fixed an issue that prevented the assessment of large source DDL files.                                          |
