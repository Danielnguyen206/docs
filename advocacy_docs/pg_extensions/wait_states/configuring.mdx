## Parameters

The following custom GUCs control the EDB Wait States extension behavior.  If you modify these parameters, reload Postgres to apply the changes.

| Parameter(s)                    | Description | Default |                                                                                                                                                                                                                                                  |
|---------------------------------|-------------|---------|
| `edb_wait_states.sampling_interval`| The interval between two EDB wait states sampling cycles.| 1 second |
| `edb_wait_states.retention_period`| Deletes EDB wait states log files after retention period. | 604800 seconds (7 days)|                                                                                                                                                                               |
| `edb_wait_states.enable_collection`| Enable or disable EDB wait states data collection. | True |                                                                                                                                                                                                                                         |
| `edb_wait_states.directory` | Stores the EDB wait states logs in this directory. The path must be a full, absolute path, not a relative path. | $PGDATA/edb_wait_states |
